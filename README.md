# bosun

Time Series Alerting Framework

# documentation

[http://bosun.org](http://bosun.org)

# usage

`bosun [-c=dev.conf] [-t]`

`-c` specifies the config file to use, defaults to `dev.conf`. `-t` parses the config file, validates it, and exits.
